# 🐍 Snake Game - My First Coding Project

Welcome! This is a classic Snake game built with JavaScript, HTML, and CSS. It's perfect for learning the basics of web development!

## How to Play

1. Click the **"Start Game"** button to begin
2. Use **Arrow Keys** to control the snake:
   - ⬆️ Up Arrow
   - ⬇️ Down Arrow
   - ⬅️ Left Arrow
   - ➡️ Right Arrow
3. Eat the red food (🔴) to grow and earn points
4. Avoid hitting the walls or yourself
5. Try to beat your high score!

## Files Explained

- **index.html** - The structure of the game (what you see)
- **style.css** - The styling and colors (how it looks)
- **script.js** - The game logic (how it works)

## How to Run

1. Go to your GitHub repository: `https://github.com/anygamez000-beep/CakeTest`
2. Click on `index.html`
3. Click the "Raw" button, then your browser will display it
4. OR: Use GitHub Pages (Settings → Pages → Deploy from main branch)
